    开始使用Octopress

    算起来已经有快两年没有写过Blog了。

    从2004年开始，到2010年写了六年Blog，而后因为毕业开始工作，慢慢就懒得写。这一次，开始使用Octopress。

    Octopress是一个基于Jekyll的静态Blog Framework，采用Rails语言，无需数据库支持，并可快速部署到Github, Heroku或自建服务器上。目前本Blog部署在 Heroku Github 上。

    Octopress使用Markdown语法发布文章，推荐使用Chen Luo开发的Markdown所见即所得编辑器MOU.

    关于Octopress的诸多好处，可见Xdite的文章(在这里)。
    如果您想使用Octopress来作为您的Blog程式且使用Heroku服务，可以参考高见龙的文章(在这里)。

    Octopress仅用来作为我个人站点的搭建程式，我仍然认为WordPress是极讚的Blog Platform，并坚持使用它。
